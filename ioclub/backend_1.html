  1. 循环控制
  • for、while
    for i in [0, 1, 2]:
    print(i)
for i in range(1, 5):
    print(i)

i = 5
while i > 0:
    print(i)
    i -= 1
  • if
    a = 1
while True:
    if a == 1:
        print(a)
        a = 2
    elif a == 2:
        print(a)
        a = 3
    else:
        break
  1. 容器

a += [4, 5, 6]

# 删除最后一个元素
last_elem = a.pop()
